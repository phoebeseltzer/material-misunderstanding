Lead
A partial record in elements.

We live in an age that claims to understand the world through replication. We build machines that learn like brains. We model memory on mycelium. We recreate photosynthesis in panels, and DNA in code.

But this is not reverence. It is simulation without context.

This work asks a simpler question: What do we really know — if we are removed from the system we’re inside?

Each entry in this archive focuses on an element: carbon, silicon, mycelium, others. Elements are supposedly universal — but we’ve built a world where their meaning depends on context, history, and access.

If we were dropped into the past, would we even know how to begin?

“Lead” was once elemental. Then industrial. Now toxic. It moved from alchemy to bullets, from piping to poison. We learned to model it. But not to live with it.

The system tried to understand itself. This is what remains.
